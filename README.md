# LGN-Bursts
Code used to create figures in Atherton et. al., 2026

1) Code used to extract trials and sessions using the Allen-SDK
2) Code used to extract burst spikes and firing rates (Figure 1)
3) Code used to decode GO trial outcome and image ID (Figure 2 a-b)
4) Code used to extract image embeddings and train and test XGBoost models (Figure2 d-g)
5) Code used to complete CSD analysis and burst rate correlations with sinks (Figure 2h-l)
